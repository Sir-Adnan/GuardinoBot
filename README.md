# Guardino-Bot

Guardino-Bot یک ربات تلگرامی برای فروش و مدیریت سرویس‌های اشتراکی VPN و Proxy است. این پروژه برای مدیرانی ساخته شده که می‌خواهند فروش، تمدید، پرداخت، مدیریت کاربران و پنل مدیریتی خود را در یک سیستم یکپارچه داشته باشند.

## امکانات اصلی

- فروش و تمدید سرویس از طریق ربات تلگرام
- پشتیبانی از کیف پول کاربر و روش‌های مختلف پرداخت
- اتصال به پنل‌های مختلف فروش و مدیریت سرویس
- پنل وب برای مدیریت کاربران، سرویس‌ها، پرداخت‌ها و تنظیمات
- پشتیبانی از چند ربات روی یک سرور
- سیستم بکاپ، آپدیت و مدیریت ساده از طریق دستور `guardino-bot`

## نصب خودکار

روش پیشنهادی نصب، استفاده از اسکریپت خودکار است:

```bash
bash <(curl -Ls --ipv4 https://raw.githubusercontent.com/Sir-Adnan/GuardinoBot/main/installer/guardino-bot.sh)
```

بعد از اجرا، منوی نصب باز می‌شود و می‌توانید پلتفرم را آماده کنید، ربات جدید بسازید، دامنه را تنظیم کنید و سرویس‌ها را مدیریت کنید.

چند دستور کاربردی بعد از نصب:

```bash
guardino-bot add main
guardino-bot list
guardino-bot update
guardino-bot logs main
guardino-bot restart main
guardino-bot backup main
guardino-bot backup-telegram
```

## نصب دستی

برای نصب دستی، ابتدا پروژه را دریافت کنید:

```bash
git clone https://github.com/Sir-Adnan/GuardinoBot.git
cd GuardinoBot
cp .env.example .env
```

سپس فایل `.env` را ویرایش کنید و حداقل موارد ضروری مثل توکن ربات، مدیران، آدرس دیتابیس، Redis و کلیدهای امنیتی را تنظیم کنید.

بعد از تنظیم فایل env:

```bash
docker compose up -d --build
```

## آپدیت

در نصب خودکار:

```bash
guardino-bot update
```

در نصب دستی:

```bash
git pull
docker compose up -d --build
```

## نکات مهم

- قبل از آپدیت یا تغییرات مهم، از اطلاعات خود بکاپ بگیرید.
- فایل `.env` شامل اطلاعات حساس است؛ آن را در اختیار دیگران قرار ندهید.
- برای استفاده عمومی و پرداخت‌ها، دامنه و HTTPS معتبر پیشنهاد می‌شود.
- این پروژه اختصاصی است و استفاده یا انتشار آن باید مطابق مجوز صاحب پروژه باشد.
